# Garten

Context-aware theming.

## Usage

```html
<script type="module">
  import Garten from './garten.js';
  Garten.init();
</script>
```

## Variables

- `--garten-accent` - Time-based accent, or special day override
- `--garten-season` - Current season accent
- `--garten-gradient` - Set during Pride month
