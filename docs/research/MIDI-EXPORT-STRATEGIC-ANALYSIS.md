### Key Industry Trends (2024-2025)

#### 1. Browser-Based Production is Maturing

**WavTool** was an AI-powered browser DAW offering chord progressions, stem separation, and audio-to-MIDI conversion. It recently shut down, highlighting the volatility of this space but also the demand.

**BandLab** now has MIDI editing, pitch correction, and mastering tools â€” blurring the line between "browser toy" and "real DAW." It has over 100 million users.

**Soundtrap** (owned by Spotify) offers unlimited tracks, built-in synths, and real-time collaboration. Soundation announced [MIDI mapping and MIDI export](https://soundation.com/station/2021/02/15/take-note-achieve-more-with-midi-mapping-and-midi-export/) as premium features specifically to help users "carry tunes to the finish line" in other DAWs.

#### 2. AI MIDI Generation is Exploding

Tools like **Staccato** (text-to-MIDI), **Lemonaide** (musical AI MIDI generator), and **HookPad's Aria** are making MIDI a central interchange format for AI-assisted composition.

> "Music producers favor plugins and native DAW features over web apps, as toggling between a web browser and your DAW rarely makes for a good experience." â€” [AudioCipher](https://www.audiocipher.com/post/ai-midi-generators)

**Implication:** MIDI export makes Keyboardia compatible with the AI music production ecosystem.

#### 3. Real-Time Collaboration Tools Are Struggling

The collaborative music space has seen significant closures:
- **Endlesss** â€” Shut down June 2024
- **Jammr** â€” Shut down (users redirected to alternatives)

> "Multiplayer DAWs and remote music collaboration apps are an emerging trend that aims to combine traditional music making with video game and contest mechanics." â€” [AudioCipher](https://www.audiocipher.com/post/multiplayer-daw-remote-music-collaboration-apps)

**Keyboardia's advantage:** Already has working real-time collaboration. Adding export would differentiate it from shuttered competitors.

#### 4. Educational Tools Don't Export â€” Intentionally?

**Chrome Music Lab** and **Ableton Learning Music** focus on exploration, not creation. They explicitly don't offer export because their goal is teaching concepts, not producing music.

**GarageBand** notably lacks MIDI export, frustrating users who want to move projects to Logic Pro. User forums consistently request this feature.

## Risks and Considerations

### Risk 1: Mission Drift

**Concern:** Adding "professional" features could dilute the "fun, simple" identity.

**Mitigation:**
- MIDI export is one button, not a complex workflow
- Core experience remains unchanged
- Export is optional â€” ephemeral users never need to see it

### Risk 2: Reduced Retention

**Concern:** Users export their MIDI and never return.

**Counter-argument:**
- Without export, users who improve leave entirely (and never return)
- With export, users can return for next ideation session
- Keyboardia becomes "starting point" in their workflow

### Risk 3: Technical Complexity

**Concern:** MIDI format is complex; bugs could frustrate users.

**Mitigation:**
- Keyboardia's data model (steps, pitches, tracks) maps cleanly to MIDI
- Standard MIDI Type 1 file format is well-documented
- Many JS libraries exist (Tone.js, jsmidgen, midi-writer-js)

### Risk 4: Competition from AI Tools

**Concern:** AI can generate MIDI; why use Keyboardia?

**Counter-argument:**
- AI generates, humans curate
- Keyboardia is for collaborative human creativity
- MIDI export makes Keyboardia compatible WITH AI workflows, not competitive against them

## Implementation Recommendations

### Phase 1: Basic MIDI Export (MVP)

**Features:**
- Export entire session as MIDI Type 1 file
- One track per Keyboardia track
- Notes include pitch, velocity, timing
- Tempo embedded in file

**Technical approach:**
```typescript
interface MIDIExport {
  format: 'midi/type1';
  tracks: {
    name: string;
    instrument: number;  // General MIDI program number
    notes: {
      pitch: number;     // MIDI note number (0-127)
      velocity: number;  // From volume p-lock
      startTick: number; // Position in MIDI ticks
      durationTicks: number;
    }[];
  }[];
  tempo: number;
  ticksPerQuarterNote: number;
}
```

**UI:**
- Single "Export MIDI" button in share menu
- Downloads `.mid` file immediately

### Phase 2: Enhanced Export Options

**Features:**
- Export single track (not just full session)
- Export selection (specific bars)
- Choose MIDI channel mapping
- Include/exclude drums on channel 10

### Phase 3: MIDI Import (Future)

**Features:**
- Drag-and-drop MIDI file onto Keyboardia
- Converts to tracks/steps
- Enables round-trip workflow with DAWs
- Enables AI-generated MIDI as input

### Phase 4: Integration with AI Tools

**Features:**
- "Send to Staccato" button
- "Remix with AI" workflow
- Collaborative AI + human ideation
