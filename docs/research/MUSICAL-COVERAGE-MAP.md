## Reading the Map

### Coverage Symbols

| Symbol | Meaning |
|--------|---------|
| `███` | **Full support today** — Genre works well with current Keyboardia |
| `░░░` | **Possible with proposed features** — Reverb, delay, instruments, triplets unlock this |
| `· ·` | **Out of reach** — Architectural limits (microtones, rubato, odd meters) block this |

### Genre Coverage Summary

| Genre | Today | With Features | Limiting Factor |
|-------|-------|---------------|-----------------|
| House/Techno | 95% | 95% | None — ideal fit |
| Disco | 90% | 90% | None |
| Synth-pop | 75% | 90% | +reverb, +delay |
| Lo-fi Hip-hop | 50% | 85% | +vinyl, +tape, +piano |
| Funk | 40% | 85% | +slap bass, +wah, +triplets |
| Soul/R&B | 35% | 80% | +Rhodes, +brass, +triplets |
| Ambient | 30% | 75% | +reverb (essential) |
| Jazz | 20% | 60% | +piano, +triplets (rubato still blocked) |
| Rock | 10% | 55% | +guitar, +drums (dynamics limited) |
| Classical | 5% | 30% | +orchestra (rubato blocked) |
| Maqam/Indian | 0% | 0% | Microtones required |

### The 35% That's Out of Reach

These remain blocked regardless of feature additions:

1. **Microtonal music** — Maqam, Indian classical, experimental
2. **Odd meter music** — 5/4, 7/8, 11/8 time signatures
3. **Rubato-dependent music** — Classical phrasing, jazz ballads
4. **Live performance** — Real-time instrument monitoring
5. **Continuous pitch** — Guitar bends, synth glides, theremin
